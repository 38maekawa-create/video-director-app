# Phase 2 計画書: 映像品質評価 + 編集支援

## 目的
Phase 1のコアエンジン（9機能、50テスト）を基盤に、映像品質評価と編集支援の機能群を実装する。
編集者がより効率的に高品質な動画を制作できるよう、品質の数値化・切り抜き支援・ハイライト支援を提供する。

## 背景
- Phase 1 完了済み: 9機能（ゲスト分類、年収演出、固有名詞フィルター、ターゲットラベリング、演出ディレクション、メンバーマスター連携、AI開発5コネクター、スプシ連携）
- Phase 3 完了済み: 5機能（config_loader、pipeline_orchestrator、file_watcher、batch_processor、pipeline_e2e）+ 100テスト
- Codex + Claude 監査済み

### ステップ2（未実装）: 映像解析機能 — 3機能

| ID | 機能名 | 概要 | 依存 |
|----|--------|------|------|
| C-1 | フレーム画像マルチモデル評価 | 代表フレームをClaude Opus 4.6 + GPT-5.4で独立評価 | opencv-python |
| C-2 | テロップ自動チェック | OCR+LLMでフォント・サイズ・配置検証 | pytesseract or easyocr |
| C-3 | 音声品質自動評価 | BGM/会話バランス + ノイズ検出 | ffmpeg |

**実装方針**: opencv-python、OCR、ffmpegの依存が必要。Phase 2のステップ2として別途実装。

### ステップ3（未実装）: ダッシュボード + FB機能 — 3機能

| ID | 機能名 | 概要 |
|----|--------|------|
| B-2 | 品質トラッキングダッシュボード | 品質スコア時系列推移 + 改善率可視化 |
| B-3 | 編集者別スキルマトリクス | 各編集者の得意/苦手を数値化 |
| NEW-3 | 編集後動画FB | 映像品質管理 + 取捨選択FB + テロップ誤字チェック |

**実装方針**: B-2はスプレッドシート or HTMLダッシュボード化。B-3はスプシの編集者データと連動。NEW-3はC-1/C-2の実装後に着手。
