## カテゴリA: ディレクションレポート自動生成（コア機能）

### A-1. ゲスト層自動分類 + 見せ方テンプレート選定
- 文字起こしからゲストの年収・職歴・年齢を自動抽出
- 層a（圧倒的に強い）/ 層b（相対的強さの言語化が必要）/ 層c（自営業家系）を自動判定
- 判定結果に応じた「見せ方テンプレート」を自動選定してレポートに反映
- **根拠**: teko_interview_direction_manual.md のゲスト分類体系

### A-2. 年収演出判断の自動化
- 年齢別早見表（20代前半:500万、20代後半:600万、30代前半:700万）に基づく自動判定
- 「強調すべき」「強調してはいけない」の二値判定をレポートに明記
- 700万以上は無条件強調フラグ
- 20代で700万以上の場合「年次・年齢補足テロップ推奨」を自動指示
- **根拠**: マニュアルの年収評価基準セクション

### A-3. 「年収以外の強さ」自動発掘
- 年収が早見表基準以下の場合、文字起こしから代替の「強さ」を自動検出
- 検出対象: 在籍企業のブランド力、キャリアパスの希少性、勤務形態の自由度、転職先の年収見込み
- ゆりかさんパターン（週4勤務500万）、りょうすけさんパターン（前職の大手ブランド）等を学習
- **根拠**: マニュアルの3-2, 3-3, 3-4セクション

### A-4. 固有名詞の自主規制判定 + 演出転換提案
- 企業名・サービス名の自動検出
- 「伏せるべきか / 出してOKか」の判定
- 伏せる場合: ピー音 + 煽りテロップ（「誰もが知る○○業界の超大手企業」）のテンプレート自動生成
- 「凄すぎて出せない」演出への自動転換提案
- **根拠**: マニュアルの固有名詞取り扱いセクション + gemini_対談動画_編集マニュアル_008.md

### A-5. ターゲット別「理想の視聴後感」チェックリスト生成
- 1層（ハイキャリア）向け: 焦燥感・所属欲求の喚起ポイントをレポートに明記
- 2層（2代目・3代目）向け: 共感・安心感の喚起ポイントをレポートに明記
- 各シーンが「どちらの層に刺さるか」のラベリング
- **根拠**: マニュアルのターゲット別視聴後感セクション

## カテゴリB: 映像品質スコアリング + トラッキング

### B-1. 7要素品質スコアリング
- カット割り（20%）、色彩・カラーグレーディング（20%）、テロップ・タイポグラフィ（15%）、BGM・サウンドデザイン（15%）、カメラワーク（10%）、構図・レイアウト（10%）、テンポ・ペーシング（10%）
- 各要素0-100点で数値化、重み付け合算で総合スコア算出
- **根拠**: teko-content-pipelineのscorer.pyパターン転用

### B-2. 品質トラッキングダッシュボード
- 動画ごとの品質スコア時系列推移
- 編集者別の品質傾向分析
- 改善率の可視化（初稿→修正1→修正2→完成版）
- **根拠**: AI開発2のパフォーマンストラッカーパターン

### B-3. 編集者別スキルマトリクス
- 各編集者の得意分野・苦手分野を数値化
- タスクアサイン時の最適マッチング提案
- 成長推移の記録
- **根拠**: member-attributes.mdの属性管理パターン

## カテゴリC: Opus 4.6 + GPT-5.4 ダブルチェック映像評価

### C-1. フレーム画像のマルチモデル評価
- 代表フレームを抽出し、Claude Opus 4.6とGPT-5.4で独立評価
- 色彩の一貫性、構図の美しさ、ブランドガイドライン適合度を両モデルで検証
- 両モデル合意の指摘のみ「ディレクション指摘」に昇格
- 不合意は「要検討」としてレポートに記録
- **根拠**: AI開発9のDualModelValidator / multi_layer_checker.pyパターン

### C-2. テロップ・タイポグラフィの自動チェック
- テロップのフォント統一性、サイズ適正、配置の一貫性を検証
- 誤字脱字の自動検出（OCR + LLM）
- ブランドカラーとの整合性チェック
- **根拠**: AI開発9のルールベース + LLMの多層チェックパターン

### C-3. 音声品質の自動評価
- BGMと会話音声のバランス検証
- ノイズレベル検出
- SE（効果音）の適切性評価
- **根拠**: AI開発4の音声処理パイプライン技術

## カテゴリD: コンテンツ戦略連動機能

### D-1. 広告CR型自動判定 + 最適化提案
- 動画コンテンツを3つのCR型（YouTube冒頭変更型 / リール・ショート型 / 実績者インタビュー型）に自動分類
- 各型に最適な構成・尺・フック設計を提案
- **根拠**: ad-creative-framework.mdの3つのCR型定義

### D-2. 冒頭フック「バズ3要素」適合チェック
- 冒頭3秒のフック部分を自動抽出
- 3要素チェック: 数字が入っているか / 「貧乏」系表現があるか / パラドックス構造があるか
- 不足要素の改善提案を自動生成
- **根拠**: ad-creative-framework.mdのバズ必須3要素

### D-3. 「池」（インサイト）カテゴリマッピング
- 動画コンテンツがどの「池」（家族・教育 / キャリア・理不尽 / 自己実現 / 子供の夢 / 親孝行 / バカンス）に属するか自動判定
- 池ごとの最適なフックテンプレート提案
- 池の使用頻度バランス（特定の池に偏っていないかの警告）
- **根拠**: ad-creative-framework.mdの池戦略

### D-4. NGパターン自動検出
- 「節約しろ」「脱サラしろ」等の禁止表現の検出
- 情緒的ポエム表現の検出
- 「副業」の前面押し出し検出
- 子供・妊娠をリスクとして扱う表現の検出
- ストーリーラインの混線検出（1本1メッセージ原則違反）
- 抽象的表現 → 具体的アイコンへの差し替え提案
- **根拠**: ad-creative-framework.mdのNG集 + 具体的アイコン変換表

### D-5. 「具体的アイコン」変換サジェスト
- 抽象的な表現を検出し、具体的アイコン（コンビニ弁当、西松屋、メルカリで値切る等）への変換を提案
- TEKO独自の変換辞書をナレッジベースとして蓄積・学習
- **根拠**: ad-creative-framework.mdのアイコン変換表
