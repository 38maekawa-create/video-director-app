# タスク指示書: VideoDirectorAgent UIリデザイン（Netflix風+高ユーザビリティ）

> 発行: 2026-03-11
> 発行者: バティ司令塔
> 実行者: 兵隊（Claude Code CLI）

## 1. 目的
映像エージェントアプリ（VideoDirectorAgent）のUIを、Netflix風のリッチなビジュアルテイストをベースに、実際の使い勝手を重視したデザインに作り直す。

## 2. 背景
- AI開発10で映像ディレクションシステムを構築中。PC版（GitHub Pages）は29件のレポートが稼働中
- スマホアプリは音声フィードバック→AI変換→Vimeoレビュー統合が核心機能
- 現在のUIプロトタイプは標準SwiftUIで「ありがちなUI」
- なおとさんから「Netflixみたいなテイストだけど、Netflixは実際超使いづらいから、UIと使用感はもっと使いやすく」とフィードバック

## 3. デザイン方針

### Netflix風の「見た目」を取り入れるポイント
- **ダークモード基調**: 真っ黒背景に映像サムネイルが映える
- **大きなサムネイル/カバー画像**: プロジェクトごとにビジュアルが目立つ
- **横スクロールのカルーセル**: カテゴリごとに横にスワイプ
- **赤系のアクセントカラー**: CTAボタン、録音中インジケーター
- **映画的なタイポグラフィ**: 太めのサンセリフ、大きい見出し
- **グラデーション背景**: サムネイルの下部にフェードグラデーション（テキスト読みやすさ確保）

### Netflixの「使いづらさ」を改善するポイント
- **自動再生OFF**: コンテンツの自動再生は絶対にしない
- **明確なナビゲーション**: 今どこにいるか常に分かる。迷子にならない
- **検索・フィルタの充実**: プロジェクト名、日付、ゲスト名で即座にフィルタ
- **ワンタップアクション**: 最も使う機能（音声FB録音）は画面のどこからでも1タップで開始
- **情報の階層が浅い**: 2タップ以内で目的の情報に到達
- **戻る動作が確実**: スワイプバック、左上の戻るボタン、どちらでも確実に戻れる
- **スクロール位置の保持**: 画面を行き来しても前の位置に戻る

## 5. 画面仕様（5画面）

### 画面1: プロジェクト一覧（ホーム）
- **ヒーローバナー**: 最新プロジェクトの大きなカバー画像（ゲスト名+撮影日をオーバーレイ）
- **カルーセル行**:
  - 「最近のフィードバック」横スクロール
  - 「全プロジェクト」横スクロール（サムネイル+ゲスト名+撮影日）
  - 「要対応」横スクロール（未完了のFBがあるもの）
- **フローティング録音ボタン**: 画面右下に常駐。どの画面からでも音声FB開始可能（赤い丸ボタン）
- サムネイルタップ→ディレクションレポートへ遷移

### 画面2: ディレクションレポート詳細
- **ヘッダー**: ゲスト名（大きく）+ 撮影日 + プロフィール（年齢・職業）
- **セクション**: タブ切替（演出ディレクション / テロップ / カメラワーク / 音声FB履歴）
- 各セクションは折りたたみ可能（Netflixの「詳細」風）
- **下部固定**: 「音声フィードバックを追加」ボタン（赤、大きめ）
- Vimeoレビューへの直リンク

### 画面3: 音声フィードバック
- **全画面モーダル**: 録音に集中できるUI
- **中央に大きな録音ボタン**: タップで録音開始、もう一度タップで停止
- 録音中: 波形アニメーション + 経過時間 + パルスする赤い枠
- 録音完了後:
  - 文字起こし結果を表示
  - AIが変換した「プロの指示」を横に並べて表示（Before/After）
  - 「この指示を送信」「やり直し」の2ボタン
- **タイムスタンプ入力**: どの時間帯のフィードバックかを指定（スライダーUI）

### 画面4: FB履歴
- **タイムライン表示**: 日付ごとにグループ化
- 各FBエントリ: 元の音声（再生ボタン）→ AIの変換結果 → 送信ステータス
- フィルタ: プロジェクト / 日付 / 未送信のみ

### 画面5: 品質ダッシュボード
- **映像品質スコア**: 大きな数字で中央表示
- **スコア推移グラフ**: 折れ線グラフ（過去10回分）
- **カテゴリ別スコア**: カメラワーク / テロップ / 音声 / 演出（レーダーチャート風）
- **改善提案**: AIからの提案リスト

### ナビゲーション
- 下部タブバー: ホーム / レポート / 録音 / 履歴 / 品質
- 録音タブは中央に配置、他より大きい赤いボタン（Instagram風）
- タブバーはNetflix風にミニマル（アイコン+小さいラベル）

## 6. UXのこだわり

### 録音体験（最重要）
- アプリ起動→録音開始まで最大2タップ
- 録音中は他の画面要素を暗く（フォーカスモード）
- 「ここがクソ」→「カット構成の再検討を推奨します。具体的には〇〇の参照映像のように...」への変換をリアルタイムプレビュー

### 情報設計
- プロジェクト一覧は「最近使った順」がデフォルト
- 検索バーは常にアクセス可能（上部固定）
- 長押しでクイックアクション（録音開始、レポート共有、Vimeo開く）

## 7. 技術仕様

- 既存のVideoDirectorAgentプロジェクト（~/AI開発10/VideoDirectorAgent/）を上書き
- Models/ は流用・拡張
- ViewModels/ は流用・拡張
- Views/ を全面書き換え
- iOS 17.0+, SwiftUI, MVVM
- モックデータで全画面動作

## 8. 完了条件

1. Xcodeで正常にビルドできる
2. 全5画面がNetflix風ダークUIで表示される
3. 録音ボタンのUIが実装されている（モック動作でOK）
4. カルーセル横スクロールが動作する
5. Before/After（音声→プロ指示変換）のUIが表示される
6. 赤+黒のカラーパレットが適用されている
7. タブバーがカスタム実装（中央録音ボタン大きめ）

## 9. 禁止事項

- 自動再生機能
- 3タップ以上必要な導線
- Netflixのロゴや商標の使用
- 映像の実再生機能（Phase 2以降）
